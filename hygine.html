Cyber Hygiene Checklist

Passwords

Use strong, unique passwords for all accounts. Consider a password manager for secure storage.

Two-Factor Authentication

Enable 2FA wherever possible to add an extra layer of security on your accounts.

Software Updates

Keep all software and apps updated to patch security vulnerabilities.

Phishing Awareness

Be cautious of suspicious emails, links, and messages. Verify sources before clicking.

Data Backup

Regularly backup important files to secure locations, both offline and cloud-based.

Antivirus Software

Install and update antivirus programs to protect against malware and ransomware.

Secure Networks

Use secure Wi-Fi connections. Avoid public, unsecured networks for sensitive tasks.

Privacy Settings

Regularly review app permissions and privacy settings to protect your personal data.

Monitor Accounts

Keep an eye on financial and social accounts for unusual activity or unauthorized access.

Stay Educated

Stay informed about the latest cyber threats and security best practices.

Recommended: Keep ID proofs, bank statements, and screenshots of incidents safe for reference.
